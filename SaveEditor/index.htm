Where are my saves?

  • %appdata%\Octogeddon
  • Steam\userdata\525620\remote

The saves are generally saved to both locations, ensure you delete the save from one location to ensure only your edited save file is loaded. You should only have a single save file present to ensure the correct save file is loaded.
